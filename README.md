# Proyecto Python con OpenCV

Este proyecto usa un entorno virtual (`virtualenv`) y requiere:

- `opencv-python`
- `numpy`
- `scipy`
- `matplotlib`

## Crear y activar entorno virtual

```bash
python3 -m venv .venv
source .venv/bin/activate
```

## Instalar dependencias

```bash
pip install -r requirements.txt
```

## Ejecutar demo

```bash
python main.py
```
